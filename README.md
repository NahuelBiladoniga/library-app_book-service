# Make Commands

* `run-prod-image`: Run Docker Compose with production settings.
* `run-dev-image`: Run Docker Compose with development settings.
* `install-air`: Install Air (Hot reload for Go).
* `run-local`: Run locally (Runs Air).

# Environment Variables

We need to set `PORT` and `SCOPE` env variables.
